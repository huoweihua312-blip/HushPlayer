## 0.5.0-beta.5 — 2026-07-21

- Git 标签：`v0.5.0-beta.5`
- 发布源码提交：`c8ed18f`
- 更新清单提交：`9a1f10d`

### 在线更新摘要

- 统一主界面视觉风格和控件层级，提升整体布局一致性
- 重新设计沉浸式歌词界面，并优化窗口打开、关闭和切换行为
- 精简侧栏导航和底部播放控制，突出常用功能入口
- 优化右侧歌词区域的自适应多行显示和当前歌词定位
- 改进不同窗口尺寸下的界面布局和歌词显示效果

### 新增

- 在公共歌曲列表中加入爱心收藏按钮，本地歌曲、在线歌曲和已保存的在线歌曲可直接加入或移出“我喜欢”，不同已加载视图之间会同步收藏状态。
- 主要侧栏导航支持拖动排序并持久化；启动时会合并有效的已保存顺序与当前默认导航项。

### 改进

- 统一主界面的视觉风格、间距和控件层级，减少重复边框与卡片嵌套，使本地歌曲列表、在线搜索结果和页面顶部区域采用更一致的布局语言。
- 重构沉浸式歌词界面，整理背景、歌词与同步设置，并优化封面、歌曲信息、歌词和播放控制的宽屏与窄屏布局。
- 优化沉浸歌词窗口的全屏切换、窗口几何恢复、移动缩放、半透明背景和退出行为。
- 精简侧栏导航与底部播放控制，重新安排当前歌曲、收藏、播放、歌词入口和音量区域，减少重复或低频入口。
- 右侧“正在播放”区域改为根据可用高度自适应显示多行歌词，并改进当前歌词定位、长歌词换行和歌曲首尾的上下文补足。
- 改进 900px 至 2560px 等不同窗口宽度以及高 DPI 环境下的主界面、播放控制和歌词显示效果。

### 性能

- 收藏成员判断复用稳定身份集合，单次收藏变化只更新相关行，避免重建整张歌曲列表。
- 沉浸歌词背景继续使用异步准备、缓存和代次校验，快速切歌时只允许最新歌曲结果更新界面。

### 修复

- 修复沉浸歌词手动浏览后的跟随恢复、单一滚动动画复用以及切歌或关闭窗口时的任务清理问题。
- 修复窄窗口下歌词被底部控制栏遮挡，以及底部时间、进度、播放和音量控件可能相互挤压的问题。

### 打包与更新

- 将应用版本更新为 `0.5.0-beta.5`，Windows 数字版本更新为 `0.5.0.5`。
- 发布 beta.5 Windows x64 更新清单，记录安装包下载地址、文件大小和 SHA-256 校验值。
- 发布前烟测进一步隔离应用数据、缓存、日志、播放队列和自定义音源目录，降低回归测试读取或改写真实用户数据的风险。

## 0.5.0-beta.4 — 2026-07-17

- Git 标签：`v0.5.0-beta.4`
- 发布源码提交：`ed3e5c5`
- 更新清单提交：`cdfe7ef`

### 在线更新摘要

- 沉浸式歌词支持自定义图片背景、模糊、暗度、透明度和填充方式调节
- 新增沉浸式歌词字体大小调节

### 修复

- 修复更新下载在 HTTP 重定向场景下因 `Content-Length` 变化而产生的文件大小误判。
- 改进 GitHub 重定向下载的兼容性，提升在线更新稳定性。

### 打包与更新

- 将应用版本更新为 `0.5.0-beta.4`，Windows 数字版本更新为 `0.5.0.4`。
- 发布 beta.4 Windows x64 更新清单，更新安装包大小、SHA-256 和下载地址。

beta.4 的旧更新清单重复列出了 beta.3 已发布的沉浸歌词外观、字号、时间偏移和背景稳定性说明；对应实现提交位于 beta.3 发布源码之前，因此这些功能只在下方 beta.3 条目中记录一次。

## 0.5.0-beta.3 — 2026-07-16

- Git 标签：`v0.5.0-beta.3`
- 发布源码提交：`596155b`
- 更新清单提交：`cb281aa`

### 在线更新摘要

- 沉浸式歌词支持自定义图片背景、模糊、暗度、透明度和填充方式调节
- 新增沉浸式歌词字体大小调节
- 新增按歌曲保存的歌词时间偏移，支持本地与在线歌曲
- 改进沉浸歌词背景渲染、切歌竞态保护和相关稳定性

### 新增

- 沉浸式歌词支持选择本地自定义背景图片，并可调节模糊、背景暗度、图片透明度和填充方式。
- 沉浸式歌词支持调整歌词字号。
- 支持为每首歌曲单独调整、保存和重置歌词时间偏移，本地歌词和在线歌词均可使用。

### 改进

- 优化沉浸式歌词背景渲染，并增加快速切歌时的结果代次校验，避免旧歌曲的封面、背景或歌词覆盖新歌曲。

### 打包与更新

- 将应用版本更新为 `0.5.0-beta.3`，Windows 数字版本更新为 `0.5.0.3`。
- 发布 beta.3 Windows x64 更新清单，记录安装包大小、SHA-256 和下载地址。

## 0.5.0-beta.1 — 2026-07-16

- Git 标签：`v0.5.0-beta.1`
- 发布源码提交：`782bf47`
- 未发现该版本对应的 `updates/beta/win-x64.json` 历史记录。

### 在线更新摘要

- 建立在线音源运行框架、自定义音源管理和统一搜索结果界面
- 在线歌曲支持收藏、远程元数据与歌词缓存，并可与本地歌曲组成混合播放队列
- 改进音乐库、搜索和在线结果的响应式界面
- 优化搜索、启动、设置加载和歌单页面的缓存与批量处理
- 调整 Windows 打包、运行时路径和用户数据隔离方式

### 新增

- 建立在线音源运行框架、自定义音源管理和统一搜索结果界面，并加入远程歌曲保存记录。
- 在线歌曲支持收藏、三态排序、远程元数据与歌词缓存，并可与本地歌曲组成混合播放队列。
- 加入在线音频缓存及相关缓存设置。

### 改进

- 重整音乐库、搜索、在线结果和歌曲详情等界面结构，并改进窄窗口与高 DPI 环境下的响应式布局。
- 分离程序资源目录与用户数据目录，完善应用数据、缓存、日志、自定义音源和运行时路径处理。

### 性能

- 为搜索加入防抖、结果缓存和增量返回处理。
- 优化启动阶段、设置加载、音乐导入、自定义歌单和“我喜欢”页面的批量处理与缓存复用。

### 修复

- 修复媒体工作线程退出生命周期、在线收藏列表稳定顺序以及缓存目录选择等问题。

### 打包与更新

- 打包内置 Node.js 运行时，并调整为 PyInstaller onedir 目录式分发。
- 加强打包路径、运行时路径与隐私边界检查，并补充可重复的干净构建验证。
